# SmartHome_DummyData
 MQTT와 DummyData를 사용한 SmartHome 프로그램입니다.

## 프로젝트 정보
### 설치
- Visual Studio, MySQL

### 사용 방법
- MySQL에서 smarthometbl 생성
- Visual Studio를 이용하여 DummyDataGen.sln 파일을 실행 
- BogusMqttWinPubApp을 먼저 실행(Ctrl + F5) 후 닫지 않고 MqttMoniteringWpfApp을 실행(Ctrl + F5)
